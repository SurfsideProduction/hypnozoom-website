Der Schutz Ihrer persönlichen Daten ist mir wichtig. Diese Datenschutzerklärung informiert Sie darüber, welche Daten beim Besuch dieser Website erhoben werden und wie sie verwendet werden. Diese Website hält die Anforderungen des Schweizerischen Datenschutzgesetzes (nDSG) ein.

Beim Aufruf der Website werden automatisch technische Daten wie IP-Adresse, Browsertyp, Betriebssystem und Uhrzeit des Zugriffs an GitHub übermittelt. Diese Daten werden von GitHub gemäss deren Datenschutzerklärung verarbeitet: GitHub Privacy Statement.

Das Kontaktformular öffnet Ihren lokalen E-Mail-Client — die Daten werden nicht über einen Server verarbeitet oder gespeichert. Eine Übermittlung findet erst durch Ihr aktives Absenden der E-Mail statt.

Ihre Daten werden nach Abschluss der Anfrage gelöscht, sofern keine gesetzliche Aufbewahrungspflicht besteht.
